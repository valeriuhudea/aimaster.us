# Media Content Romania with Next.js & Tailwind CSS

## How to use

### 1\. Clone this Repository

```bash
git clone https://github.com/valeriuhudea/mediacontent-dev.git
```

### 2\. Navigate to the directory

```
cd mediacontent-dev
```

### 3\. Install dependencies

```
yarn install
# or
npm install
```

### 5\. Run Dev Server

```
yarn dev
# or
npm run dev
```
